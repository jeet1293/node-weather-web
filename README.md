# node-weather
Node app to get weather forecast based on location provided by user

## Getting started:

App requires [Node.js](https://nodejs.org/) v12+ to run.

```
$ npm install
```

Run the server:

```
$ node app
```

Open a browser at the following URL:

```
http://localhost:3300/
```
